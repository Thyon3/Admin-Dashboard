# Next Admin Dashboard

A modern admin dashboard built with Next.js, TypeScript, and Tailwind CSS.

## Features

- 🚀 **Next.js 16** with App Router
- 🎨 **Tailwind CSS v4** for styling
- 📱 **Responsive Design** for all screen sizes
- 🌙 **Dark Mode** support
- 🔧 **TypeScript** for type safety
- 🧩 **Reusable Components** with comprehensive UI library
- 📊 **Data Tables** with pagination
- 🎯 **Forms** with validation and state management
- 🔔 **Alerts** and notifications
- 🖼️ **Image** and video components

## Tech Stack

- **Framework**: Next.js 16
- **Language**: TypeScript
- **Styling**: Tailwind CSS v4
- **Code Quality**: ESLint, Prettier
- **Build Tool**: Webpack/Turbopack

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

```
src/
├── app/                 # Next.js App Router pages
├── components/          # Reusable UI components
│   ├── ui/            # Base UI components
│   ├── form/          # Form components
│   ├── tables/        # Table components
│   └── common/        # Common components
├── context/            # React contexts
├── hooks/              # Custom React hooks
├── icons/              # SVG icons
└── layout/             # Layout components
```

## UI Components

The project includes a comprehensive set of UI components:

- **Buttons**: Multiple variants and sizes
- **Alerts**: Success, error, warning, info variants
- **Badges**: Status indicators and labels
- **Avatars**: Image and text-based avatars with status
- **Modals**: Regular and fullscreen modals
- **Forms**: Input fields, checkboxes, switches
- **Tables**: Data tables with pagination
- **Images**: Responsive image grids
- **Videos**: YouTube embed components
